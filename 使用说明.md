# 📝 PDF 签字工具使用说明

## 🚀 快速开始

### 第一步：在多维表格添加记录

1. 打开多维表格：https://my.feishu.cn/base/XRnnbOKv4aMHgFsdsR3cdBIjnej
2. 添加一行记录：
   - **文档名称**：填写文档名称（如"测试合同"）
   - **原始PDF**：上传需要签字的 PDF 文件

### 第二步：生成签字链接

使用命令行生成链接：

```bash
cd ~/.openclaw/workspace/skills/feishu-pdf-signature

python3 scripts/generate_link.py <记录ID> <PDF下载地址>
```

**示例：**
```bash
python3 scripts/generate_link.py recxxxxxxxxxxxx https://example.com/test.pdf
```

### 第三步：签字

把生成的链接发给签字人，打开后：
1. 预览 PDF 文档
2. 在右侧手写签字
3. 点击「提交签字」

### 第四步：查看结果

签字完成后，会自动更新多维表格：
- **已签字PDF**：自动填入签字后的文件
- **签字人**：记录签字人（可选）
- **签字时间**：记录签字时间（可选）

---

## 📋 字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 文档名称 | 文本 | 主键，文档名称 |
| 原始PDF | 附件 | 待签字的 PDF |
| 签字链接 | 文本 | 生成的签字链接（可手动填入） |
| 已签字PDF | 附件 | 签字完成后的 PDF |
| 签字人 | 文本 | 记录签字人姓名 |
| 签字时间 | 日期 | 记录签字时间 |

---

## 🔧 进阶使用

### API 调用方式

```bash
curl -X POST http://localhost:5000/api/generate-link \
  -H "Content-Type: application/json" \
  -d '{
    "record_id": "记录ID",
    "pdf_url": "PDF下载地址"
  }'
```

### 配置页面

访问：http://localhost:5000/config

可以修改多维表格连接配置。

---

## ⚠️ 注意事项

1. **PDF 下载地址**：需要是可以直接访问的公开链接
2. **一次性链接**：每个链接只能使用一次，签字后立即失效
3. **服务状态**：确保 Flask 服务在运行中（http://localhost:5000）

## 🆘 问题排查

### 服务未启动？

```bash
cd ~/.openclaw/workspace/skills/feishu-pdf-signature
./scripts/start.sh
```

### 端口被占用？

```bash
# 杀掉占用 5000 端口的进程
lsof -ti:5000 | xargs -r kill -9
```
